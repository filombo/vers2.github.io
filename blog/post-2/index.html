How to create a site using HUGO

Published on Apr 01, 2021 by Filippo Mauri

Want to have a site similar to mine without spending a dime? Do you want it a little more beautiful? Then you’re in the right place! A few days ago I had a few hours free during an afternoon and I wanted to try my hand at what I thought was one of the greatest feats in human history. Well, thank goodness, it wasn’t as hard as I thought it would be. Even though you need to use the terminal you don’t have to be scared. To create the site we will use Hugo , which as it says on their site, is The world’s fastest framework for building websites. Instead to make your super nice site available to everyone we will use Github Pages.

Download Hugo and create your first website

Let’s start by downloading this framework. To do this you will need to have Homebrew which is easily downloadable (via terminal) from their site.

After successfully downloading Homebrew just paste this string in the terminal.

$ brew install hugo

In this way the download will start and tend to end, if you are lucky in a very short time. Now you’re ready to create your first site! Now I’ll need to, via the terminal, point to the desktop by typing:

$ cd
$ cd Desktop

Once you press enter you will see before the “$” the name of the folder where you are that is “Desktop”. At this point you just type the command below replacing “websitename” with the name you want to give to your site.

$ hugo new site websitename

At this point, on the terminal will appear a message confirming that you have created your site.

Choice of theme and customization

One of the first things you want to do is to change the theme with which you want to display your site. Hugo gives you a big hand in this. In fact in the themes section of their site, you will find a lot of wonderful themes but above all totally free. In the page relating to the theme you can, by pressing the “demo” button have a preview of the site so you can get an idea of how it could be yours at the end of this tutorial.

Once you have chosen the theme you will need to download it. To do this enter the folder of your site by typing:

$ cd websitename

Then enter the themes folder:

$ cd themes

At this point go to the Github page of the theme you have chosen by pressing the Download button on the theme page. At this point you will open a Github page in which there is a green button where it says “code”. Pressing it will appear a dropdown from which you must copy the proposed link. Now write in the terminal:

$ git clone themelink

Paste instead of themelink the link you copied using the key combination CTRL + Shift + V.

At this point open the file named “config.toml” contained in the folder of your site. In the first lines there will be written: theme=“default” Just replace default with the name of your theme. Once you have saved the file your site will be ready in its basic structure.

To view it will be enough, after having typed in terminal:

$ cd ..

to return to the folder of the site, type:

$ hugo server

Once you press enter a message will appear saying that you can visit the site at the address “localhost:1313”. To view your site go to this address. In this way, however, the site will be visible only to you if you keep on the server that you started with the previous command.

Github Pages configuration

To make sure that everyone can see your beautiful site you have to create a Github repo named in this way: username.github.io I recommend not to choose stupid username for the Github account.

Once created, go back to the terminal and press CTRL + c to stop the server. To create files that can actually be interpreted as sites by Github you’ll need to type in the terminal:

$ hugo

At this point in the folder of your site you will have created a folder named “public”.

The only thing missing is to copy in the Github repo the contents of that folder. Once you press “Commit” in the file upload pages you have to go to the settings section and scroll down until you find the Github Pages option. If everything went well you should find a green banner where it’s written the link where you can find yours.

Pressing that link you’ll find your site!

That’s it!
